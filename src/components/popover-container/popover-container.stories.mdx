# PopoverContainer

## Quick Start

To use Popover Container, import the `PopoverContainer` and pass the content as children.

```jsx
import React from 'react';
import PopoverContainer from "carbon-react/lib/components/popover-container";

const MyComponent = () => (
  <PopoverContainer
    title='Hello world!'
  />
);
```
