### Custom pageSizeSelectionOptions;
Shows 1, 2 or 5 items per page.

<Preview>
  <Story name="page-size-selection-options" parameters={{ info: { disable: true }}} >

      onPagination={ () => {} }
      totalRecords={ 100 }
      showPageSizeSelection={ true }
      pageSizeSelectionOptions={

          { id: '1', name: 1 },
          { id: '2', name: 2 },

        ]
      }
      pageSize={ 1 }
    />
  </Story>
</Preview>
